# PLC Conveyor Belt Sorting System

Simple PLC automation project simulating a conveyor belt system that sorts objects based on sensor detection.

## System Overview

Objects move on a conveyor belt.  
A sensor detects the object and identifies whether it is metal or plastic.

The PLC then activates the corresponding actuator to push the object into the correct chute.

## Inputs

Start button  
Stop button  
Object detection sensor  
Metal detection sensor  

## Outputs

Conveyor motor  
Metal sorting actuator  
Plastic sorting actuator  

## Skills Demonstrated

PLC logic design  
Industrial automation concepts  
Sensor-based sorting systems
